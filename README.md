# Telco Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn data to identify key factors influencing customer retention and business performance.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI (for dashboard visualization)

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Exploratory Data Analysis (EDA)
- Identifying churn patterns based on:
  - Contract type
  - Monthly charges
  - Tenure
  - Payment methods

## 📈 Business Insights
- Customers with month-to-month contracts have higher churn rates
- Higher monthly charges correlate with increased churn
- Long-term customers are less likely to churn

## 📷 Dashboard
Power BI dashboard is included for visualization of churn trends and KPIs.

## 🎯 What This Project Demonstrates
- Data analysis and problem-solving
- Python-based data processing
- Business insight generation from raw data
- Dashboard reporting for decision-making

## 🚀 How to Run
1. Open the notebook file in Jupyter
2. Run all cells step by step
3. Review outputs and insights
